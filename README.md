# 🎮 Simon Says Game

A simple and interactive Simon Says Game built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo
https://simon-says-game-8sku.onrender.com

## 📌 Features
- Interactive gameplay
- Random color sequence generation
- Score tracking
- Game Over and Restart functionality
- Responsive design
- Works on both desktop and mobile devices

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)


## 🎯 How to Play
1. Open the game.
2. Press any key (Desktop) or tap/click anywhere (Mobile/Desktop) to start.
3. Watch the color sequence carefully.
4. Repeat the sequence by clicking the colored buttons.
5. The sequence gets longer with each level.
6. The game ends when you click the wrong button.
